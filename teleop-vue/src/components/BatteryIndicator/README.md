# BatteryIndicator

## Usage
To modify the style, apply a CSS class to the component. Otherwise, the default style will be used.

### Example:
```html
<battery-indicator class="battery-component" />
```
```css
.battery-component {
    gap: 10%;
    height: 1.5rem;
    width: 2.5rem;
    --fill-color: white;
    --border-color: white;
    --font-size: 80%;
}
```
